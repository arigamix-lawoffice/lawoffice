Requirements for extensions compilation:
1. Installed Visual Studio 2022 version 17.6 or later. Its possible to use free Community Edition.
2. If IDE is used, then install .NET SDK 6.0.x of last available version: https://dotnet.microsoft.com/download/dotnet/6.0

For deploying web service on IIS its required to install .NET Runtime & Hosting Bundle version 6.0.16 or later:
https://dotnet.microsoft.com/download/dotnet/6.0
